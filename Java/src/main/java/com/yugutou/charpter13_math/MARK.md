## 一个数的幂
一个数可以分解为1b1 + 2b2 + 4b3 + 8b4这种形式
1 2 3 4为该数的二进制位的幂，b1 b2 b3 b4为该数的二进制位。
## 最大公约数
 a, b, c  a > b, a % b = c gcd(a,b) = gcd(b,c)
## 素数
1. 判断素数，从2开始到根号n判断
2. 判断n以内有多少素数，埃氏筛。如果一个数是素数，那么2x,3x,4x,5x一定不是素数
## 丑数 一个数只有2, 3或5三个因子
1. 判断丑数，依次和2，3，5运算
2. 判断第n个丑数，埃氏筛。如果一个数是丑数，那么2x,3x,5x都是丑数。可以使用优先队列或者三指针。
